Random Forest — How It Works

Random Forest builds multiple decision trees on random data subsets, then aggregates their predictions by majority vote. This ensemble approach reduces overfitting and improves generalisation compared to a single decision tree.

Logistic Regression — How It Works

Logistic Regression models the probability of heart disease using the sigmoid function: P = 1 / (1 + e⁻ˣ). It maps input features to a probability between 0 and 1, classifying as positive if P > 0.5.
